# 🏡 Australian Real Estate Price Prediction

## 📌 Project Overview

This project focuses on building an end-to-end Machine Learning pipeline capable of accurately predicting housing prices in the Australian real estate market.

The project was developed as part of a business expansion strategy for **Surprise Housing**, a US-based real estate company planning to enter the Australian housing market. The primary goal is to help the company identify profitable investment opportunities using predictive analytics and data-driven decision making.

The solution leverages:
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Regression Modeling
- Model Evaluation
- Business Insight Extraction

---

# 🎯 Business Objective

The objective of this project is to:

✅ Predict property prices accurately  
✅ Identify important variables affecting house prices  
✅ Assist real estate investment decisions  
✅ Reduce investment risk  
✅ Maximize return on investment (ROI)

---

# 📊 Dataset Information

The dataset contains:

- **1460 training records**
- **81 variables/features**
- Numerical and categorical attributes
- Real-world missing values and outliers

### Dataset Includes:
- Property size
- Neighborhood
- Basement quality
- Garage details
- Construction year
- House condition
- Interior quality
- Sale price
