# Australian Real Estate Price Prediction

## Project Overview

This project focuses on predicting house prices in the Australian real estate market using Machine Learning techniques.

The objective is to help Surprise Housing identify profitable investment opportunities by accurately forecasting property prices and understanding the key factors affecting housing values.

---

## Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Multiple Machine Learning Models
- Hyperparameter Tuning
- Model Evaluation
- Feature Importance Analysis

---

## Models Used

- Linear Regression
- Ridge Regression
- Lasso Regression
- ElasticNet
- Random Forest Regressor

---

## Evaluation Metrics

- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
